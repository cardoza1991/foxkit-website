1. Introduction

Cardoza Services LLC ("we," "our," or "us") operates the FoxKit SMS Security mobile application (the "Service"). This Privacy Policy explains how we collect, use, and protect your information when you use our Service.

Our Commitment: FoxKit is designed with privacy-first principles. We minimize data collection and maximize your control over your personal information.

2. Information We Collect

2.1 SMS Messages (Local Processing Only)

  • What: Incoming SMS message content, sender information, timestamps
  • Purpose: Real-time threat analysis and protection
  • Storage: Processed locally on your device - NOT stored or transmitted
  • Retention: 0 days - Messages are analyzed and immediately discarded

2.2 Threat Detection Metadata

  • What: Anonymized threat indicators (malicious URLs, phone number patterns)
  • Purpose: Improve security detection capabilities
  • Storage: Local device only, no cloud transmission
  • Data: No personal content, only security fingerprints

2.3 App Usage Analytics (Optional)

  • What: App performance metrics, crash reports, feature usage
  • Purpose: Improve app functionality and user experience
  • Control: You can opt-out in Settings
  • Data: No SMS content, only technical metrics

2.4 Device Information

  • What: Device model, Android version, app version
  • Purpose: Compatibility and support
  • Storage: Local only

3. How We Use Your Information

3.1 Primary Purposes

  • SMS Threat Detection: Analyze incoming messages for phishing, scams, and malicious content
  • Real-time Protection: Block or warn about suspicious SMS messages
  • Security Updates: Update threat detection rules and patterns

FoxKit is not intended for children under 13. We do not knowingly collect personal information from children under 13. If you believe we have inadvertently collected such information, please contact us immediately.

8. Technical Implementation

8.1 Permission Usage

  • READ_SMS: Analyze incoming messages for threats
  • RECEIVE_SMS: Intercept messages before delivery for protection
  • NOTIFICATIONS: Alert you to blocked threats

8.2 Data Minimization

  • Processing Time: <100 milliseconds per message
  • Memory Usage: Messages cleared immediately after analysis
  • Storage: Zero persistent storage of message content
  • Network: No network permissions - fully offline
